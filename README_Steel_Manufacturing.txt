UNDERSTANDING THE MANUFACTURING CONTEXT
---------------------------------------

Steel Production Flow (for context only, not tested):
Raw Materials → Blast Furnace (molten iron) → Steel Making Shop/SMS (steel) → 
Continuous Casting Machine/CCM (solid slabs) → Rolling Mill (shaped products) → Finishing

Common Terms You'll Encounter:
- Heat: A batch of molten steel (like a production batch)
- Ladle: Large vessel for holding molten steel
- Tundish: Intermediate vessel in continuous casting
- Refractory: Heat-resistant lining material
- SOP: Standard Operating Procedure
- WBS: Work Breakdown Structure
- PPE: Personal Protective Equipment
- CCM: Continuous Casting Machine
- SMS: Steel Making Shop
- BF: Blast Furnace

You don't need to memorise these — just helpful context for understanding the documents.
